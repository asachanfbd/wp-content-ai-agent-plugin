A basic WordPress plugin boilerplate for an AI content agent.

== Description ==

This is a boilerplate for the WP Content AI Agent plugin. It serves as a starting point for building AI-powered content features in WordPress.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-content-ai-agent` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings link to configure the plugin.

== Frequently Asked Questions ==

= Does this plugin require an API key? =

This is a boilerplate, so currently it does not connect to any service. Future versions will likely require an API key.
